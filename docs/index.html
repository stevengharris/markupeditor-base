Welcome to the MarkupEditor

The MarkupEditor supports WYSIWYG editing in a web view. The "source of truth" for the MarkupEditor is standard HTML, styled using standard CSS.

The MarkupEditor aims to support the same functionality as GitHub flavored Markdown. However, instead of dealing with the distractions in Markdown-based text and the mysteries of how that text is translated to HTML and presented, the MarkupEditor always presents the HTML directly as you edit it.
