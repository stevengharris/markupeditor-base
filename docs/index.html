Welcome to the MarkupEditor

The MarkupEditor supports WYSIWYG editing in a web view. The "source of truth" for the MarkupEditor is standard HTML, styled using standard CSS.

Like Markdown, the MarkupEditor keeps the focus on what you're writing, with a minimum of distractions. Like Markdown, it supports just enough functionality to help you organize and format your writing to get your points across effectively. The MarkupEditor's WYSIWYG approach means you see what you're writing in realtime, instead of dealing with the distractions of composing Markdown and the uncertainty of how that Markdown is translated to HTML and presented.
